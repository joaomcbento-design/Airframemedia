01

Responsável pelo Tratamento de Dados

AirFrame Media  ·  info@airframemedia.pt  ·  +351 969 442 101

02

Que dados recolhemos?

Recolhemos apenas os dados que nos fornece diretamente, nomeadamente:

  • Nome e endereço de e-mail, quando preenche o formulário de contacto ou nos envia uma mensagem;
  • Número de telefone, caso nos contacte por essa via;
  • Qualquer outra informação que decida partilhar connosco no âmbito de um pedido ou projeto.

Recolhemos também dados de utilização de forma agregada e anónima, que não permitem a sua identificação pessoal.

03

Para que fins utilizamos os seus dados?

Os seus dados são utilizados exclusivamente para:

  • Responder às suas questões ou pedidos de orçamento;
  • Gerir a relação contratual decorrente de serviços prestados;
  • Melhorar o nosso website e a experiência do utilizador (dados de análise agregados).

Não utilizamos os seus dados para fins de marketing direto sem o seu consentimento explícito, nem os partilhamos com terceiros para fins comerciais.

04

Base legal para o tratamento

O tratamento dos seus dados baseia-se em:

  • Execução de contrato — quando os dados são necessários para a prestação de um serviço solicitado;
  • Consentimento — quando nos contacta voluntariamente através do formulário ou nos fornece os seus dados para fins específicos;
  • Interesses legítimos — para análise agregada do desempenho do website.

05

Durante quanto tempo guardamos os seus dados?

Os dados recolhidos através do formulário de contacto são conservados pelo período necessário para responder ao seu pedido e, caso resulte numa relação contratual, pelo prazo legalmente exigido (em regra, 10 anos para documentação contabilística). Dados que não deram origem a qualquer relação contratual são eliminados ao fim de 12 meses.

06

Os seus direitos

Ao abrigo do Regulamento Geral sobre a Proteção de Dados (RGPD), tem direito a:

  • Aceder aos seus dados pessoais;
  • Solicitar a retificação de dados incorretos;
  • Solicitar o apagamento dos seus dados ("direito ao esquecimento");
  • Opor-se ao tratamento ou solicitar a sua limitação;
  • Apresentar reclamação à autoridade de controlo nacional, a CNPD (www.cnpd.pt).

Para exercer qualquer um destes direitos, contacte-nos através de info@airframemedia.pt.

07

Segurança dos dados

O website airframemedia.pt é alojado no GitHub Pages (GitHub, Inc.), que implementa medidas técnicas e organizativas adequadas para proteger os dados contra acesso não autorizado, perda ou destruição. Para mais informações, consulte a política de privacidade em github.com.

08

Alterações a esta política

Reservamo-nos o direito de atualizar esta política sempre que necessário. A data de "última atualização" no topo do documento refletirá sempre a versão mais recente.
